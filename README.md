# Mobile-Based Public Transport Safety Reporting System

This repository contains a full-stack project scaffold for a public transport safety reporting system.

## Structure

- `client/` – React + Vite frontend with PWA support
- `server/` – Node.js + Express backend with MongoDB/Mongoose

## Getting Started

1. Install client dependencies:
   - `cd client`
   - `npm install`

2. Install server dependencies:
   - `cd server`
   - `npm install`

3. Add environment variables in `server/.env`.

4. Run the backend:
   - `npm run dev`

5. Run the frontend:
   - `npm run dev`
